ngx-form-draft Demo

This demo shows auto-save and restore of form drafts using ngx-form-draft.

  • Reactive form — FormGroup + FormArray with all common input types
  • Template-driven form — ngForm with the same variety of inputs

Both forms use [draftDebounce]="500" and [draftShowOnChange]="true". Fill a form, refresh or navigate away, then come back to see the draft restored.
